# История изменений SV Converter

## Версия 2.1 (2026-06-26)

### Новые возможности

#### 1. Автоматическое подтягивание пары файлов
- При выборе `.cfg` файла автоматически ищется и загружается `.dat` файл с таким же именем
- При выборе `.dat` файла автоматически ищется и загружается `.cfg` файл с таким же именем
- Экономит время при работе с большим количеством осциллограмм

#### 2. Новая вкладка "5. Просмотр данных"
Полноценный инструмент анализа сконвертированных данных:

**Режимы просмотра:**
- **Осциллограммы** — графики токов и напряжений во времени
  - Раздельные графики для токов (Ia, Ib, Ic, In) и напряжений (Ua, Ub, Uc, Un)
  - Интерактивный зум и панорамирование (панель инструментов matplotlib)
  - Сохранение графиков в PNG/SVG

- **Векторная диаграмма** — фазорное представление
  - Визуализация действующих значений и фазовых углов
  - Раздельное отображение токов (сплошные линии) и напряжений (пунктир)
  - Автоматическое масштабирование для удобства просмотра

- **Частота** — анализ частоты сигнала
  - Расчет мгновенной частоты по нулевым переходам
  - Сравнение с номинальной частотой
  - Отображение средней частоты
  - Зум для детального анализа отклонений

- **Таблица значений** — числовые данные
  - Все 8 каналов в табличном виде
  - Прокрутка по вертикали и горизонтали
  - Оптимизированное отображение (не более 1000 строк)
  - Статистика: количество сэмплов, частота дискретизации, длительность

#### 3. Кнопка "Просмотр" в 4-й вкладке
- Быстрый переход к просмотру данных после конвертации
- Кнопка активируется автоматически после успешной конвертации

### Технические улучшения
- Добавлена зависимость от `matplotlib` и `numpy` для графиков
- Graceful degradation: если matplotlib не установлен, программа работает без функции просмотра
- Парсинг данных напрямую из DAT файла (без повторного чтения PCAP)
- Применение всех коэффициентов трансформации (Ктт, Ктн, K3i0, K3u0) при отображении

### Установка зависимостей

```bash
pip install -r requirements.txt
```

Или вручную:
```bash
pip install matplotlib>=3.5.0 numpy>=1.21.0
```

### Использование новых функций

1. **Автоподтягивание файлов:**
   - Выберите `.cfg` файл → `.dat` загрузится автоматически (если существует)
   - Или наоборот: выберите `.dat` → `.cfg` загрузится автоматически

2. **Просмотр данных:**
   - Выполните конвертацию (вкладка 4)
   - Нажмите кнопку "📊 Просмотр данных"
   - Или перейдите на вкладку "5. Просмотр данных" вручную
   - Выберите нужный режим просмотра (Осциллограммы/Векторная диаграмма/Частота/Таблица)

3. **Работа с графиками:**
   - Используйте панель инструментов matplotlib для зума и панорамирования
   - Кнопка "Home" сбрасывает зум
   - Кнопка "Pan" — перетаскивание графика
   - Кнопка "Zoom" — выделение области для увеличения
   - Кнопка "Save" — сохранение графика в файл
