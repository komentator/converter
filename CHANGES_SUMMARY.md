### ✅ 5. Парсинг данных для просмотра
**Файлы:** `gui_main.py` (строки 444-487)

**Метод:** `parse_samples_from_dat()`

**Что делает:**
1. Парсит DAT файл через `parse_dat_ascii()`
2. Извлекает значения по каждому каналу
3. Применяет масштабирование из CFG (mult, offset)
4. Применяет коэффициенты трансформации (Ктт, Ктн, K3i0, K3u0)
5. Сохраняет в `self.parsed_samples` для отображения

**Структура данных:**
```python
{
    'time': [0.0, 0.00025, 0.0005, ...],
    'sample_rate': 4000.0,
    'line_freq': 50.0,
    'params': {...},
    'Ia': [123.4, 125.1, ...],
    'Ib': [...],
    ...
}
```

## Технические особенности

### Graceful degradation
Если matplotlib не установлен, программа:
- Показывает предупреждение на 5-й вкладке
- Остальной функционал работает нормально
- Кнопка "Просмотр" остается доступной (но показывает сообщение об ошибке)

### Производительность
- Таблица показывает не более 1000 строк (каждый N-й сэмпл)
- Графики используют встроенную оптимизацию matplotlib
- Парсинг данных происходит один раз при конвертации

### Безопасность типов
- Проверки на наличие данных перед отрисовкой
- Обработка исключений при парсинге
- Защита от деления на ноль при расчете частоты
