# Быстрый старт - SV Converter v2.1

## Установка

```bash
pip install matplotlib numpy
```

## Запуск

```bash
python main.py
```

## Использование в 5 шагов

### 1️⃣ Загрузка файлов
- Выберите `.cfg` файл → `.dat` загрузится автоматически
- Или выберите `.dat` → `.cfg` загрузится автоматически

### 2️⃣ Проверка каналов
- Нажмите "Автоматическое определение"
- Или настройте вручную через "Редактировать"

### 3️⃣ Параметры (можно оставить по умолчанию)
- MAC: `01-0C-CD-04-00-01`
- APPID: `4000`
- svID: `RET61850_SV1`
- Коэффициенты: Ктт=1000, Ктн=1100

### 4️⃣ Конвертация
- Нажмите "▶ Конвертировать в PCAP"
- Дождитесь завершения

### 5️⃣ Просмотр результата
**Сохранить файл:**
- Нажмите "⬇ Сохранить PCAP"

**Или посмотреть графики:**
- Нажмите "📊 Просмотр данных"
- Выберите режим:
  - **Осциллограммы** — токи и напряжения во времени (с зумом!)
  - **Векторная диаграмма** — фазорное представление
  - **Частота** — анализ частоты сигнала
  - **Таблица** — числовые значения

## Работа с графиками

**Зум:**
1. Нажмите кнопку 🔍 (Zoom) на панели инструментов
2. Выделите область мышью
3. Кнопка 🏠 (Home) сбрасывает зум

**Панорамирование:**
1. Нажмите кнопку ✋ (Pan)
2. Перетаскивайте график мышью

**Сохранение:**
- Нажмите 💾 (Save) для сохранения графика в PNG/SVG

## Горячие клавиши matplotlib

- `Home` / `r` — сброс зума
- `Left arrow` / `c` — назад
- `Right arrow` / `v` — вперед
- `p` — режим Pan
- `o` — режим Zoom
- `s` — сохранить график
- `g` — сетка вкл/выкл
- `l` / `k` — логарифмическая шкала X / Y

## Советы

✅ **Автоподтягивание файлов** — называйте `.cfg` и `.dat` одинаково
✅ **Зум на графиках** — выделяйте интересную область для детального анализа
✅ **Векторная диаграмма** — для быстрой оценки симметрии фаз
✅ **Частота** — для обнаружения провалов/всплесков частоты при КЗ
✅ **Таблица** — когда нужны точные числовые значения

## Troubleshooting

**Ошибка: ModuleNotFoundError: No module named 'matplotlib'**
```bash
pip install matplotlib numpy
```

**Графики не отображаются**
- Проверьте, что конвертация завершена успешно
- Убедитесь, что matplotlib установлен

**Автоподтягивание не работает**
- Убедитесь, что `.cfg` и `.dat` файлы имеют одинаковое имя
- Файлы должны быть в одной папке
