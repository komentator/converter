## 🎯 ВЫБЕРИ СВОЙ ПУТЬ

### Путь 1️⃣: Я хочу СРАЗУ начать (без EXE)
```bash
cd sv_converter
python main.py
# или на macOS/Linux:
python3 main.py
```
✅ Браузер откроется автоматически  
✅ Начните конвертировать сразу  
⏱️ Время: 30 секунд

---

### Путь 2️⃣: Я хочу собрать EXE (Windows)

#### 2A - Самый простой способ (РЕКОМЕНДУЕТСЯ)
```bash
cd sv_converter
python build_exe.py
```
✅ Скрипт все сделает автоматически  
✅ Установит PyInstaller  
✅ Соберет EXE  
⏱️ Время: 3-5 минут (первый раз), затем 1-2 минуты

#### 2B - Через batch файл (для новичков)
1. Откройте папку `sv_converter` в проводнике
2. Дважды кликните `build_exe.bat`
3. Ждите сборки
✅ EXE будет в `dist\SV_Converter\SV_Converter.exe`

#### 2C - Ручная сборка (полный контроль)
```bash
pip install pyinstaller
cd sv_converter
pyinstaller sv_converter.spec
```

## 📊 ПАРАМЕТРЫ (по умолчанию)

| Параметр | Значение | Описание |
|----------|----------|---------|
| MAC | 01-0C-CD-04-00-01 | MAC адрес целевого устройства |
| APPID | 4000 | ID приложения (hex) |
| VLANID | 0 | ID виртуальной сети (0 = без VLAN) |
| svID | RET61850_SV1 | Идентификатор потока |
| confRev | 1 | Версия конфигурации |
| Simulation | включен | Флаг тестовых данных |
| Ктт | 1000 | Коэффициент трансформации тока |
| Ктн | 1100 | Коэффициент трансформации напряжения |
| K3i0 | 1000 | Коэффициент нейтрального тока |
| K3u0 | 1905.2 | Коэффициент нейтрального напряжения |

## 🔍 ПРОВЕРКА РЕЗУЛЬТАТА

### В Wireshark
```
1. Откройте Wireshark (https://www.wireshark.org/)
2. File → Open → выберите sampled_values.pcap
3. Фильтр: eth.dst == 01:0c:cd:04:00:01
4. Смотрите структуру кадра:
   - Destination MAC
   - APPID: 0x4000
   - svID: "RET61850_SV1"
   - Значения каналов (в hex)
```

### Сравнение с эталоном
```
Если у вас есть эталонный PCAP дампа:
1. Откройте оба файла в Wireshark
2. Сравните первый кадр
3. Должны совпадать:
   - MAC адреса
   - APPID
   - svID
   - Simulation флаг
   - Значения каналов
```
