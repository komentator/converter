# 🔨 Как собрать SV Converter в EXE файл (Windows)

## Способ 1: Автоматический (рекомендуется)

### Вариант A: Через Python скрипт (проще всего)

1. **Откройте папку `sv_converter`** (где находятся файлы проекта)

2. **Запустите в этой папке командную строку** (cmd.exe):
   - На Windows 10/11: Shift + правый клик → "Открыть окно PowerShell здесь"
   - Или: Windows + R → введите `cmd` → нажмите Enter
   - Затем: `cd путь\к\sv_converter`

3. **Выполните команду:**
   ```cmd
   python build_exe.py
   ```
   
4. **Ждите** 2-3 минуты (скрипт установит PyInstaller и соберёт EXE)

5. **Готово!** EXE будет в папке `dist\SV_Converter\SV_Converter.exe`

### Вариант B: Через batch скрипт

1. **Откройте папку `sv_converter`**

2. **Дважды кликните** на файл `build_exe.bat`

3. **Скрипт автоматически:**
   - Установит PyInstaller
   - Соберёт EXE
   - Покажет путь к результату

4. **Готово!** EXE будет в папке `dist\SV_Converter\`

## Способ 2: Ручная сборка

### Требования

- **Python 3.7+** (скачать с https://www.python.org/)
- **PyInstaller** (установится командой ниже)

### Пошагово

1. **Проверьте Python:**
   ```cmd
   p![img.png](img.png)
   ```
   Должно быть 3.7 или выше.

2. **Установите PyInstaller:**
   ```cmd
   pip install pyinstaller
   ```

3. **Перейдите в папку проекта:**
   ```cmd
   cd путь\к\sv_converter
   ```

4. **Соберите EXE:**
   ```cmd
   pyinstaller sv_converter.spec
   ```

5. **Ждите 2-3 минуты** (идёт сборка)

6. **EXE готов:** 
   ```
   dist\SV_Converter\SV_Converter.exe
   ```

## Варианты сборки

### Вариант 1: Папка с файлами (по умолчанию)
```cmd
pyinstaller sv_converter.spec
```
**Результат:**
- Папка: `dist/SV_Converter/` (~100+ MB)
- Главный файл: `SV_Converter.exe` (~5 MB)
- Много других файлов (Python, библиотеки)

**Плюсы:** Быстрая сборка, меньший exe  
**Минусы:** Много файлов

### Вариант 2: Один файл (однофайловый EXE)
```cmd
pyinstaller sv_converter.spec --onefile
```
**Результат:**
- Один файл: `dist/SV_Converter.exe` (~50-60 MB)
- Больше ничего не нужно

**Плюсы:** Один файл, легко распространять  
**Минусы:** Медленнее собирается, первый запуск медленнее

## Проблемы и решения

### Проблема: "Python не найден"
```
'python' is not recognized
```

**Решение:**
1. Скачайте Python с https://www.python.org/downloads/
2. **При установке отметьте:** "Add Python to PATH"
3. Перезагрузитесь
4. Попробуйте снова

### Проблема: "PyInstaller не установился"
```
ERROR: Could not find a version that satisfies the requirement
```

**Решение:**
```cmd
pip install --upgrade pip
pip install pyinstaller
```

### Проблема: "sv_converter.spec не найден"
```
FileNotFoundError: [Errno 2] No such file or directory: 'sv_converter.spec'
```

**Решение:**
- Убедитесь, что находитесь в папке `sv_converter`
- Проверьте, что файл `sv_converter.spec` есть в этой папке

### Проблема: "Антивирус блокирует"
Некоторые антивирусы блокируют PyInstaller. 

**Решение:**
- Добавьте папку проекта в исключения антивируса
- Попробуйте отключить антивирус на время сборки
- Или используйте вариант сборки "--onefile"

## Использование собранного EXE

### Запуск
```cmd
dist\SV_Converter\SV_Converter.exe
```

Или просто **дважды кликните** на файл.

### Первый запуск
- Может быть медленнее (распаковка файлов)
- Это нормально
- Последующие запуски будут быстрее

### Распространение
Если собрали **вариант 1 (папка)**:
- Скопируйте всю папку `dist/SV_Converter/`
- Раздайте другим пользователям

Если собрали **вариант 2 (однофайловый)**:
- Скопируйте только файл `dist/SV_Converter.exe`
- Раздайте другим пользователям

## После сборки

### Проверить EXE
1. Откройте `dist\SV_Converter\SV_Converter.exe`
2. Браузер должен открыться автоматически
3. Вы должны видеть веб-интерфейс программы

### Создать ярлык (для удобства)
1. Правый клик на `SV_Converter.exe`
2. Выберите "Отправить" → "На рабочий стол (ярлык)"
3. Теперь иконка будет на рабочем столе

### Удалить временные файлы
```cmd
rmdir /s /q build
del /q *.egg-info
```
