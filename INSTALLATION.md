# Инструкция по установке и сборке SV Converter

## Windows

### Вариант 1: Запуск из исходного кода (для разработки)

#### Требования
- Python 3.7 или выше
- Git (опционально)

#### Установка

1. Скачайте проект:
```bash
# Если у вас установлен Git
git clone <repo-url>
cd sv_converter

# Или распакуйте ZIP архив
# и перейдите в папку проекта
```

2. Проверьте версию Python:
```bash
python --version
```
Должно быть Python 3.7 или выше.

3. Запустите приложение:
```bash
python main.py
```

Или используйте файл `run.bat`:
```bash
run.bat
```

### Вариант 2: Сборка и запуск .EXE (для конечных пользователей)

#### Требования
- Python 3.7 или выше
- PyInstaller

#### Сборка

1. Установите PyInstaller:
```bash
pip install pyinstaller
```

2. Перейдите в папку проекта и выполните сборку:
```bash
cd sv_converter
pyinstaller sv_converter.spec
```

3. Готовый файл находится в `dist/SV_Converter/`:
```
dist/
└── SV_Converter/
    └── SV_Converter.exe
```

#### Запуск EXE

Просто двойным кликом откройте `SV_Converter.exe`. Приложение автоматически:
1. Запустит локальный сервер на http://127.0.0.1:8000
2. Откроет браузер с интерфейсом программы
3. Будет готово к конвертации файлов

#### Создание однофайлового EXE (опционально)

Для создания одного EXE файла вместо папки:
```bash
pyinstaller sv_converter.spec --onefile
```

Результат: `dist/SV_Converter.exe` (большой файл ~50+ МБ)

### Установка в системное меню (опционально)

Для удобства можно создать ярлык:
1. Правый клик на `SV_Converter.exe`
2. "Отправить" → "На рабочий стол (ярлык)"
3. Теперь иконка будет на рабочем столе

## macOS

### Требования
- Python 3.7 или выше (установить можно через Homebrew)
- Homebrew (рекомендуется)

### Установка и запуск

1. Установите Python (если нет):
```bash
brew install python3
```

2. Скачайте проект:
```bash
git clone <repo-url>
cd sv_converter
```

3. Запустите приложение:
```bash
bash run.sh
```

Или напрямую:
```bash
python3 main.py
```

### Сборка приложения (опционально)

Для распространения среди других пользователей macOS:

1. Установите PyInstaller:
```bash
pip3 install pyinstaller
```

2. Выполните сборку:
```bash
pyinstaller sv_converter.spec
```

3. Создайте DMG образ (опционально):
```bash

# Создание DMG
create-dmg --volname "SV Converter" \
  dist/SV_Converter.dmg dist/SV_Converter/
```

## Проблемы и решения

### Проблема: "Python не найден" на Windows
**Решение:** 
- Переустановите Python и отметьте опцию "Add Python to PATH"
- Или используйте полный путь: `C:\Python39\python.exe main.py`

### Проблема: "ModuleNotFoundError" при запуске
**Решение:**
- Убедитесь, что вы находитесь в папке проекта
- Проверьте, что все файлы (converter.py, server.py, main.py) на месте
- Проверьте наличие папки `static/` с файлом `index.html`

### Проблема: "Port 8000 already in use"
**Решение:**
- Закройте другие приложения, использующие порт 8000
- Или отредактируйте main.py и измените port на другой (например, 8001)

### Проблема: Браузер не открывается автоматически
**Решение:**
- Откройте браузер вручную и перейдите на http://127.0.0.1:8000/
- Проверьте, что сервер запустился (должно быть сообщение в консоли)

### Проблема: Файлы .cfg/.dat не загружаются
**Решение:**
- Убедитесь, что файлы имеют правильное расширение (.cfg, .dat)
- Проверьте, что это подлинные файлы COMTRADE (не переименованные)
- Откройте файлы текстовым редактором и проверьте формат

### Проблема: Конвертация не работает / Error при нажатии на конвертировать
**Решение:**
1. Убедитесь, что все каналы сопоставлены (выпадающие списки не пусты)
2. Проверьте правильность ввода параметров (MAC, APPID и т.д.)
3. Посмотрите ошибку в консоли/статусе интерфейса
4. Проверьте, что файлы .cfg и .dat корректны

## Разработка

### Структура проекта

```
sv_converter/
├── main.py              # Точка входа
├── converter.py         # Основная логика конвертации
├── server.py           # HTTP сервер
├── static/
│   └── index.html      # Веб-интерфейс
├── sv_converter.spec   # Конфигурация PyInstaller
├── run.bat            # Быстрый запуск (Windows)
├── run.sh             # Быстрый запуск (Linux/macOS)
├── README.md          # Основная документация
└── INSTALLATION.md    # Этот файл
```

### Запуск в режиме разработки

1. Установите требования (опционально):
```bash
pip install -r requirements-dev.txt
```

2. Запустите сервер с автоперезагрузкой (требует watchdog):
```bash
python main.py
```

Или для быстрого цикла тестирования:
```bash
python -m pdb main.py
```

### Модификация интерфейса

Все изменения интерфейса находятся в `static/index.html`:
- HTML структура в теге `<body>`
- CSS стили в теге `<style>`
- JavaScript логика в теге `<script>`

Обновляйте браузер (Ctrl+F5) для перезагрузки кешированных файлов.

### Добавление новых параметров SV

1. Добавьте поле в HTML (static/index.html)
2. Добавьте чтение этого поля в функцию `doConvert()` 
3. Передайте параметр в `params` при отправке на сервер
4. Обработайте параметр в функции `convert()` (converter.py)
