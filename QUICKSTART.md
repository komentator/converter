# 🚀 Быстрый старт (5 минут)

## Для Windows (с готовым EXE)

1. **Скачайте** файл `SV_Converter.exe` из папки `dist/`
2. **Запустите** кликом
3. Браузер откроется автоматически на http://127.0.0.1:8000

## Для разработчиков (Python)

```bash
# 1. Откройте командную строку/терминал в папке проекта
cd sv_converter

# 2. Запустите на Windows:
python main.py

# 3. Или на macOS/Linux:
python3 main.py

# 4. Браузер откроется автоматически
```

## Использование (все одно и то же)

1. **Загрузите** файлы
   - Выберите .cfg файл
   - Выберите .dat файл
   
2. **Нажмите** "🔍 Автоматическое определение"

3. **Проверьте** значения параметров (обычно уже заполнены)

4. **Нажмите** "▶ Конвертировать в PCAP"

5. **Скачайте** результат кнопкой "⬇ Скачать файл .pcap"

**Готово!** Файл `sampled_values.pcap` готов к использованию.

---

## Проблемы?

- **Файлы не загружаются**: проверьте расширения (.cfg, .dat)
- **Конвертация не работает**: убедитесь, что все каналы выбраны в выпадающих списках
- **Браузер не открывается**: откройте вручную http://127.0.0.1:8000/
- **Port 8000 занят**: измените номер порта в `main.py` на 8001
