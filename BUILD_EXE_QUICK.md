# 🔨 Сборка SV Converter в EXE для Windows

## ⚡ Быстрая сборка (выбери один способ)

### Способ 1: Автоматический (самый простой)

**Требуется:** Python 3.7+ установлен

```bash
# 1. Откройте командную строку в папке sv_converter
cd путь\к\sv_converter

# 2. Выполните одну команду:
python build_exe.py

# 3. Ждите 2-3 минуты (скрипт установит PyInstaller и соберёт EXE)

# 4. EXE готов в: dist\SV_Converter\SV_Converter.exe
```

### Способ 2: Через batch скрипт (для Windows)

**Требуется:** Python 3.7+ установлен

1. **Откройте папку `sv_converter`** в проводнике
2. **Дважды кликните** на файл `build_exe.bat`
3. **Ждите 2-3 минуты** (скрипт автоматически установит PyInstaller и соберёт)
4. **EXE готов** в папке `dist\SV_Converter\SV_Converter.exe`

### Способ 3: Ручная сборка (максимальный контроль)

```bash
# 1. Установите PyInstaller (если не установлен):
pip install pyinstaller

# 2. Перейдите в папку проекта:
cd путь\к\sv_converter

# 3. Соберите EXE:
pyinstaller sv_converter.spec

# 4. Ждите 2-3 минуты

# 5. EXE готов в: dist\SV_Converter\SV_Converter.exe
```

## 🐛 Проблемы при сборке

### Ошибка: "Python не найден"
```
'python' is not recognized as an internal or external command
```

**Решение:**
1. Скачайте Python с https://www.python.org/downloads/
2. **При установке** отметьте "Add Python to PATH"
3. Перезагрузитесь
4. Откройте заново командную строку

### Ошибка: "PyInstaller не установился"
```
ERROR: Could not find a version that satisfies the requirement pyinstaller
```

**Решение:**
```bash
pip install --upgrade pip
pip install pyinstaller
```

### Ошибка: "sv_converter.spec не найден"
```
FileNotFoundError: sv_converter.spec
```

**Решение:**
- Убедитесь, что находитесь в папке `sv_converter`
- Проверьте, что файл `sv_converter.spec` есть в этой папке
- Используйте полный путь: `cd C:\Users\YourName\Downloads\sv_converter`

### Ошибка: "Недостаточно места на диске"
```
No space left on device
```

**Решение:**
- Очистите диск (сборка требует ~500 MB временного места)
- Удалите старые сборки: `rmdir /s /q build dist`
